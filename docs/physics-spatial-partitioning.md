# Physics Spatial Partitioning

## 比較

### uniform grid

- 固定範囲をあらかじめ決め、その範囲を均一セルで分割する方式。
- 範囲内の密な配置では実装と走査が単純。
- 初期化時にワールド範囲を決める必要があり、疎な配置や広範囲では空セルの扱いが増えやすい。

### spatial hash

- AABB が重なるセル座標をハッシュ化し、使用されたセルだけを保持する方式。
- ワールド範囲を固定しなくても扱えるため、疎な配置や広範囲の2D当たり判定に向く。
- ハッシュと重複ペア除去のコストはあるが、初期スコープでは固定範囲を持たない方が扱いやすい。

## 初期方式

初期方式は spatial hash とする。

理由:

- Physics2D の初期 API に固定ワールド範囲を要求しない。
- 使用セルだけを保持できるため、広範囲に散らばった対象でも不要なセルを確保しない。
- 同じセルに入った AABB だけを候補ペアとして返せるため、全組み合わせより探索対象を削減できる。
